Analyse en composante principale : ACP
* Objectif : 
    - reduire la dimension des donnees
    - visualiser les donnees
* Principe :
    - on cherche les directions dans lesquelles les donnees sont les plus variables
    - on projette les donnees dans ces directions
* Methode :
    - on cherche les vecteurs propres de la matrice de covariance
    - on projette les donnees dans les directions des vecteurs propres
* Application :
    - on peut visualiser les donnees
    - on peut reduire la dimension des donnees
* Probleme :
    - on perd de l'information
    - on perd de l'information

    - on a des donnees en 3D
    - on cherche les directions dans lesquelles les donnees sont les plus variables
    - on projette les donnees dans ces directions
    - on obtient des donnees en 2D
